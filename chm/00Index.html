Collection of a set of miscellaneous, useful functions

Help pages for package ‘sundry’ version 0.4-2

angular Applies the angular transformation to proportions.
angular.mod Applies the modified angular transformation to a vector of counts.
efficiencies Computes the nonzero canonical efficiency factors between two projection matrices
ginv Computes the generalized inverse of a rectangular matrix
info.crit Computes AIC and BIC for a model.
is.zero Tests whether all elements are zero.
mean.harmonic Calcuates the harmonic mean.
mean.op computes the mean operator matrix corresponding to a (generalized) factor.
plot.profiles Produces a trellis graph in which the values for an individual unit are plotted.
projdf Computes the degrees of freedom or number of nonzero canonical efficiency factors of a projection matrix
projectnonorth Computes the projection and Residual operators for two projection matrices that may be nonorthogonal
randrep generates patterned uniform random numbers.
reml.lrt Performs REML likelihood ratio test.
